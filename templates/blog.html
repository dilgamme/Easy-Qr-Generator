About QR   Home

Full form of QR is Quick Response. It was proposed in 1994 by Japanese barcode developer company denso wave.
QR code typically transforms any kind of digital information into a 2D matrix like shape with black square entries in white background.
Typical QR code is shown in the below diagram. It can store any information into this 2D grid. Therefore, it can be used anywhere when some digital contractual information processing is required.
Applications can be product tracking, item identification, time tracking, location tracking, document management, general marketing and many more.

There are mainly two types of QR:
1. Static QR code
2. Dynamic QR code

Static QR code can store information only which do not change over time period. It is one-time storage. For example, phone number, SMS, Email etc. QR code which store these kind of information are static as these information do not alter over time. When someone other person scans this QR code, the device will understand the information stored in the scanned QR code.

Dynamic QR codes are those which store dynamic information such is location tracking, product shippibg status etc.
QR code is a very easy way from which we can store as much complex information as we want in it.
